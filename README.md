# VkBotKit bot template

## Get Started

### Install
```sh
pip3 install https://github.com/kensoi/vkbotkit/tarball/dev
git clone https://github.com/kensoi/vkbotkit-bot-template
cd ./
```

### Insert your token
```sh
export VKBOTKIT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
```

### Start bot
```sh
python3 app.py
```
